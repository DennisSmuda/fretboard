# Fretboard

Fretboard Canvas Display Engine for tonal & tonal-fretboard

![screenshot](src/assets/screen.png)

## Usage

```javascript
let fretboard = new Fretboard({
  tuning: 'guitar',
  scale: 'Bb lydian',
  numFrets: 15,
  numStrings: 6
})
```

#### Install

- yarn install
- yarn run build
- yarn start
